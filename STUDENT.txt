Cara menjalankan aplikasi:
1. Install dependencies: npm install
2. Development mode: npm run start-dev
3. Build production: npm run build

Fitur aplikasi:
1. Registrasi dan login pengguna
2. Melihat daftar cerita
3. Melihat detail cerita
4. Menambahkan cerita baru dengan kamera
5. Menentukan lokasi cerita menggunakan peta
6. Melihat lokasi cerita di peta

Troubleshooting tips:
- Jika kamera tidak berfungsi, pastikan memberikan izin kamera di browser
- Jika peta tidak muncul, periksa koneksi internet dan reload halaman
- Pastikan aplikasi dijalankan di browser modern (Chrome, Firefox, Edge terbaru)
